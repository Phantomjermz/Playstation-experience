SSD Storage: The PS5 is equipped with an ultra-fast SSD (Solid State Drive) that allows for faster loading times, so games can start more quickly and levels can load faster. In order to match the expected speed of the console’s internal storage (and the expansion drive slot in the console), you’ll need an M.2 solid state drive with a high read speed. An M.2 drive is a form of NVMe (Non-Volitile Memory Express), which connects to the consoles via PCIe, which makes it much faster than a standard Solid State drive. M.2 modules are also generally much smaller than standard hard drives — they look more like a stick of RAM than a standard SSD.
